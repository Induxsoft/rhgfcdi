 # Rhgfcdi (Dotación de insumos al personal).
El paquete Rhgfcdi (Dotación de insumos al personal) de V12 es el módulo que permite controlar y administrar la entrega de insumos, materiales y recursos a los empleados, asegurando trazabilidad y orden en la distribución. Su propósito es garantizar que cada dotación quede registrada, vinculada al trabajador correspondiente y reflejada en los procesos administrativos del ERP.
